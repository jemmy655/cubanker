Welcome to the CommunityBanker BIGBOOK

Community Banker is an open source banking system for use by credit unions and cooperative banks which offers the promise of a free (as in beer) yet fully capable and compliant small banking system. It does not try and re-invent the wheel in that it is designed to be utilised with an existing core accounting package such as Pastel, GnuCash or AccPac etc.

The package is known as cubanker, is written in PHP & MySQL and utilises the CodeIgniter framework.

This BIGBOOK aims to provide information about:

  • Concepts : Basic Cooperative Banking concepts
  • Cooperative Banking : An introduction to Cooperative Banking
  • Installation : Installing the System
  • User Manual : Complete User Manual
  • The Software: Design Documentation & Programmer Notes
