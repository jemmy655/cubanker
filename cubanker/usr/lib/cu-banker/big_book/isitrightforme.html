Is it right for me?

cubanker is right for you if:

  • You want a no-fuss open-source credit union system.
  • You need exceptional performance.
  • You need broad compatibility with accounting packages.
  • You want a system that requires nearly zero configuration.
  • You want a credit union system that does not require you to use the command line.
  • You want a credit union system that is so easy to install it can be hosted anywhere.
  • You eschew complexity, favoring simple solutions.
  • You need clear, thorough documentation.
